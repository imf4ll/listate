&nbsp;
## 🔧 Setup:
- Clone this repository:<br>
`git clone https://github.com/z3oxs/listate` or Download ZIP and unzip;<br><br>
- Move to repository:<br>
`cd listate`<br><br>
- Install all NodeJS dependencies:<br>
`npm install` or `yarn install`<br><br>
- Run:<br>
`yarn start`<br>
